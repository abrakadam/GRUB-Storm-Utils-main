# GRUB Storm Utils

Универсальная программа для управления загрузчиком GRUB и системной информацией. Поддерживает Linux и Windows, предоставляет графический интерфейс, детальную информацию о системе и помощь в установке двойной загрузки.

## Платформы

**Linux и Windows** - кроссплатформенная утилита:
- **Linux**: Полное управление GRUB с графическим интерфейсом
- **Windows**: Подготовка к установке Linux и информация о системе

## Быстрый старт

### Windows
1. Скачайте `GRUBStormUtils.exe`
2. Запустите двойным кликом
3. Выберите нужный пункт меню

### Linux
```bash
# Автоматическая установка
cd Linux && ./install.sh && sudo grub-storm

# Или ручной запуск
cd Linux/build && ./GRUBStormUniversal
```

- **Подготовка системы** к двойной загрузке

- **Рекомендации** по выбору дистрибутива
- **Проверка** готовности системы

### Встроенный терминал

- Выполнение любых системных команд
- Прямой доступ к shell
- Просмотр результатов выполнения команд
- **Встроенные команды для управления GRUB:**
  - `help` - Показать справку по командам
  - `sysinfo` - Информация о системе
  - `grub` - Информация о GRUB
  - `entries` - Список всех записей
  - `default` - Текущая запись по умолчанию
  - `version` - Версия GRUB
  - `config` - Конфигурация GRUB
  - `set-default <name>` - Установить запись по умолчанию
  - `add-entry <title>` - Добавить запись
  - `timeout <seconds>` - Установить таймаут
  - `update-grub` - Обновить GRUB
  - `backup` - Создать резервную копию
  - `restore` - Восстановить из резервной копии
  - `check` - Проверить установку GRUB
  - `install <device>` - Установить GRUB на устройство
  - `memtest` - Добавить запись Memtest86+
  - `recovery` - Добавить режим восстановления

## Установка

### Linux (Автоматическая установка)

**Универсальный установщик для всех дистрибутивов:**

```bash
# Скачайте и распакуйте архив
cd GRUB-Storm-Utils-main/Linux

# Запустите установочный скрипт
./install.sh

# Для удаления программы
./install.sh uninstall
```

**Поддерживаемые дистрибутивы:**
- Ubuntu/Debian/Linux Mint (apt-get)
- Fedora/CentOS/RHEL (dnf/yum)
- Arch Linux/Manjaro (pacman)
- openSUSE (zypper)
- Gentoo (emerge)

### Linux (Ручная установка)

**Ubuntu/Debian:**
```bash
sudo apt-get update
sudo apt-get install build-essential cmake pkg-config
mkdir build && cd build
g++ -o GRUBStormUniversal ../src/universal_main.cpp
sudo cp GRUBStormUniversal /usr/local/bin/grub-storm
```

**Fedora/CentOS:**
```bash
sudo dnf install gcc-c++ cmake pkg-config
mkdir build && cd build
g++ -o GRUBStormUniversal ../src/universal_main.cpp
sudo cp GRUBStormUniversal /usr/local/bin/grub-storm
```

**Arch Linux:**
```bash
sudo pacman -S base-devel cmake pkgconf
mkdir build && cd build
g++ -o GRUBStormUniversal ../src/universal_main.cpp
sudo cp GRUBStormUniversal /usr/local/bin/grub-storm
```

## Использование

### Linux

**Универсальная версия (рекомендуется):**
```bash
# Запуск с правами root (полная функциональность)
sudo grub-storm

# Запуск без прав (ограниченная функциональность)
grub-storm

# Прямой запуск
./build/GRUBStormUniversal
```

**Доступные версии:**
1. **GRUBStormUniversal** - Универсальная версия с автоопределением GUI
2. **GRUBStormGUI** - Графическая версия (требует Zenity)
3. **GRUBStormGTK** - GTK3 версия (требует libgtk-3-dev)
4. **GRUBStormUtils** - Текстовая версия

**Автоматическое определение интерфейса:**
- Если есть Zenity → Графический интерфейс
- Если есть Dialog → Текстовый GUI
- Иначе → Текстовый режим

### Windows

**Запуск программы:**
```bash
# Через командную строку
GRUBStormUtils.exe

# Через проводник
Двойной клик по GRUBStormUtils.exe
```

**Требования:**
- Windows 10/11
- GTK3 библиотеки (включены в сборку)
- Права администратора для полного функционала

### Интерфейс

**Вкладки:**
1. **System Info** - Полная информация о системе
2. **GRUB Info** - Информация о GRUB и конфигурация (включает: версия, статус, конфигурация, скрипты, модули, шрифты, темы, разделы, EFI, UUID, mount info)
3. **Entries** - Таблица записей загрузки с управлением
4. **Terminal** - Встроенный терминал с командами управления GRUB и приглашением user@hostname:~$
5. **Help** - Инструкция по использованию и меры предосторожности
6. **Operations** - Операции с GRUB (продвинутая кастомизация)

**Кнопки управления:**
- **Refresh** - Обновить список записей (на вкладке Entries)
- **Execute** - Выполнить команду в терминале
- **Apply** - Установить запись по умолчанию (на вкладке Operations)
- **Add Entry** - Добавить новую запись (на вкладке Operations)
- **Set Timeout** - Изменить таймаут загрузки (на вкладке Operations)
- **Update GRUB** - Запустить update-grub (на вкладке Operations)
- **Apply Advanced Settings** - Применить продвинутые настройки (на вкладке Operations)

### Вкладка Operations

Операции с GRUB и продвинутая кастомизация:

**Основные операции:**
- Установка записи по умолчанию (выпадающий список)
- Добавление новых записей загрузки (title, kernel, initrd)
- Изменение таймаута загрузки (счетчик 0-300 секунд)
- Обновление GRUB (кнопка update-grub)

**Продвинутая кастомизация GRUB:**
- Параметры ядра (kernel parameters)
- Графический режим (graphics mode)
- Разрешение загрузки
- Включение/отключение splash screen
- Режим тихой загрузки (quiet boot)
- Применение продвинутых настроек

### Вкладка Help

Детальная инструкция по использованию программы:
- Инструкция по использованию всех вкладок
- Меры предосторожности и рекомендации
- Что делать если система не загружается
- Советы по безопасности

## Детальная информация о системе

Программа автоматически определяет:
- **ОС**: Linux/Windows
- **Версия ОС**: Детальная версия
- **Версия ядра**: Текущая версия ядра
- **Архитектура**: x86, x86_64, ARM и т.д.
- **Битность**: 32 или 64 бит
- **Дистрибутив**: Ubuntu, Debian, Fedora и т.д.
- **Хостнейм**: Имя компьютера
- **Пользователь**: Текущий пользователь
- **CPU**: Модель и количество ядер
- **Память**: Общий и доступный объем

## Предупреждения

- **Опасный режим** позволяет модифицировать конфигурацию GRUB
- Неправильные изменения могут сделать систему незагружаемой
- Программа автоматически создает резервные копии перед изменениями
- Рекомендуется иметь Live USB с Linux для восстановления
